Helper toolbox to better integrate W3TC and other features of the ApisCP hosting platform.

== Description ==

Toolbox for ApisCP is a helper plugin to better integrate your blog in an [ApisCP hosting platform](https://apiscp.com/) environment.

Major features in the Toolbox include:

* [Admin] **Secure Access Tokens** on login to bypass mod_evasive filtering
* [WP CLI] W3TC Environment Rewrite Rules fix and cleanup utility
* [WP CLI] W3TC Rewrite Rules check control utility

== Installation ==

1. Install and activate [w3-total-cache](https://wordpress.org/plugins/w3-total-cache/) plugin first.
2. Install and activate the Toolbox for ApisCP.
3. Simply start using it with WP CLI.
